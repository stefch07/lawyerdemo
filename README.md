# Harrington Legal — Lawyer Demo

A premium, responsive law-firm consultation website created as a showcase project for a web-design agency.

**Live demo:** [lawyerdemo.stefanoscharal505525.chatgpt.site](https://lawyerdemo.stefanoscharal505525.chatgpt.site)

## Highlights

- Editorial boutique-law-firm visual direction
- Responsive desktop, tablet, and mobile layouts
- Six clearly structured practice areas
- Firm story, values, and client approach sections
- Accessible navigation and keyboard focus states
- Consultation form with validation and a confirmation state
- Reduced-motion support
- Original generated architectural hero imagery

## Technology

- React 19
- Next.js 16
- TypeScript
- Tailwind CSS 4
- Vinext and Vite

## Local development

Node.js 22.13 or newer is required.

```bash
npm install
npm run dev
```

Then open the local address printed in the terminal.

## Production build

```bash
npm run build
```

## Project structure

```text
app/page.tsx         Main website and interactions
app/globals.css      Visual system and responsive styling
app/layout.tsx       Metadata and root layout
public/              Static assets and hero imagery
```

## Disclaimer

Harrington Legal is a fictional firm created for demonstration purposes. The contact details and content in this repository are not associated with a real legal practice.
